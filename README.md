## Play the game

- Run `main` to play the game (takes time to intialize.

- Or `pip install pygame` (only if you haven't installed) and then run `main.py` in an IDE to play the game.

### Instruction:

Play: Click `play` button or press `p`.

Move: `←↑→↓` arrow keys

Shoot: `Space` bar

Quit: `q`
